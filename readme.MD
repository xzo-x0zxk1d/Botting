# 📘 README

## ⚠️ Important  
This tool should only be used in ways that follow the Terms of Service of any platform it interacts with.  
Do **not** use automation to harm communities or violate platform rules.

---

## 📦 Installation

1. Click the **Code** button on this repository.  
2. Select **Local → Download ZIP**.  
3. After downloading, extract the ZIP file to a folder on your device.  
4. Open the extracted folder — **do not move files out of it**.  
5. Run **`botting.exe`**.

---

## ▶️ Usage

Once the program starts:

1. You will be prompted to enter a **Server ID**.  
2. Input the ID of the server you want the tool to interact with.  
3. Follow the on‑screen instructions provided by the application.

---

## 🔍 How to Get a Server or User ID

To enable Developer Mode in Discord:

1. Open **User Settings**.  
2. Scroll to **Advanced**.  
3. Enable **Developer Mode**.  
4. Right‑click any server, user, or channel → **Copy ID**.
